24/02/2017 Project 1 - Week 0
  • Project Name: ExplorerElli (working title)
  • Project type: Unity Game Project
  • Time Frame: 1 month (24/02/2017 - 24/03/2017)

My first month long challenge, my aim is to create a Unity game where the player gets to run around on an outsized 3d object like Mario in Super Mario Galaxy. For now, I'm aiming to create a functioning gravity system (that draws and rotates the player towards the centre of the object) and to create a couple of sample objects for the player to use. This project should test my mathematical physics knowledge. It will also give me the opportunity to learn more about blender/3d modelling and importing these models into Unity as well as creating custom meshes.
